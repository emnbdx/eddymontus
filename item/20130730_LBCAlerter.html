Dans mon activité, je fais pas mal de recherche sur le bon coin et il est souvent frustrant d'appeler pour une annonce trop tard.

J'ai alors développé un système d'alerte pour le site le bon coin.
Y ayant passé beaucoup de temps cette application est une version de démonstration limitée dans les fonctionnalités et dans le temps.
Il est possible d'acheter l'application, cet achat correspond à un mois d'utilisation (à renouveler donc) ou de souscrire à un abonnement.

Je prévois pas mal d'amélioration de cette application, notamment le style, mais certainement la migration vers un site web. Ce qui permettrait le support des téléphones mobiles.
Cette application nécessite le Framework .Net 4.5
